## 분석

### 1. 전부 Baseline 3(현재 프로덕션)를 이겼다
최하위 학습모델(Linear QR, 609.7)조차 Baseline 3(712.6)보다 낫고, 최상위(QRF onehot, 408.5)는 **약 43% 개선**. Phase 4에서 세운 가설("동/면적 정보를 규제하며 반영하면 KDE를 이길 수 있다")이 확인됨.

### 2. One-hot이 Target Encoding을 일관되게(근소하게) 이김
트리 계열 4개(QRF/LightGBM/CatBoost/XGBoost) 전부에서 onehot 버전이 target_enc 버전보다 Pinball이 낮음(차이는 0.3~4.3 정도로 크진 않음). 동이 54개뿐이라 원핫으로도 차원이 크게 부담되지 않았고, 트리 모델이 개별 동 컬럼을 직접 분할 조건으로 쓸 수 있어 James-Stein 축소보다 정보 손실이 적었던 것으로 해석됨. **→ 최종 모델은 onehot 인코딩 채택.**

### 3. Pinball과 Coverage는 다른 이야기다
Pinball 기준 1~3위(QRF/LightGBM/CatBoost 전부 onehot)가 근소한 차이(408.5~410.8)라 사실상 동급인데, **Coverage@90은 크게 갈린다**:
- QRF: 0.924~0.927 — 이상적인 0.90보다 살짝 높음(약간 넓게 잡음, 보수적)
- CatBoost: 0.886~0.889 — 0.90에 가장 근접
- LightGBM: **0.866~0.869 — 뚜렷한 과소보정(90% 구간이라면서 실제로 87%밖에 못 담음)**

Pinball 최적화만으로는 이 문제가 안 보인다 — 구간 예측을 실제 서비스(예: "90% 신뢰구간")에 쓰려면 Coverage도 같이 봐야 함.

### 4. 최종 후보: QRF (onehot)
Pinball 1위(408.5)이면서 Coverage도 가장 준수(0.924)해서 종합적으로 가장 안전한 선택. LightGBM(onehot)이 Pinball은 근소하게 앞서지만(409.6, QRF와 차이 0.3%) Coverage가 크게 못 미쳐(0.866) **채택하지 않음**.

### 5. Linear QR과 NGBoost는 왜 처졌나
Linear QR(609.7)은 baseline 3보다 조금 나은 정도 — 면적·연식·지역과 보증금의 관계가 트리 모델이 잡아내는 비선형/상호작용 패턴을 선형모델이 못 잡는다는 뜻. NGBoost(438.7)도 트리 4형제보다 처지는데, 얕은 기본 트리(depth=3) 기반의 분포 파라미터 추정 방식이 이 데이터의 트리 분할 패턴을 충분히 못 살린 것으로 추정.

## 5-Fold CV 안정성 확인

스크립트: `training/phase5_cv_stability.py` — 튜닝된 하이퍼파라미터(n_estimators=363, max_depth=14, min_samples_leaf=10)를 고정하고 **train 내에서** 5-fold 랜덤 분할로 재검증(재튜닝 아님, 분산만 확인).

| Fold | Pinball | CRPS_근사 | Coverage@90 | MAE |
|---|---|---|---|---|
| 1 | 466.7 | 933.4 | 0.921 | 1,509.4 |
| 2 | 453.8 | 907.6 | 0.919 | 1,459.1 |
| 3 | 457.3 | 914.5 | 0.919 | 1,460.6 |
| 4 | 443.3 | 886.6 | 0.924 | 1,430.7 |
| 5 | 461.5 | 923.1 | 0.918 | 1,482.4 |
| **평균±표준편차** | **456.5 ± 8.8** | 913.0 ± 17.7 | **0.920 ± 0.002** | 1,468.5 ± 29.3 |

**Pinball 표준편차가 평균의 약 2%, Coverage@90은 5개 폴드 전부 0.918~0.924로 사실상 고정** — 특정 분할에서만 우연히 잘 나온 게 아니라 하이퍼파라미터가 안정적으로 작동함을 확인.

CV의 Pinball(456.5)이 원래 val 평가(408.5)보다 다소 높게 나오는데, 이는 모델이 나빠서가 아니라 **CV는 train 기간(2023.08~2025.05, 전세비율 30.7%로 상대적으로 높고 고액 전세 비중이 큰 시기) 내에서만 평가하고, val은 그보다 이후(전세비율 22.0%로 낮아진) 기간이라 애초에 두 평가 대상의 데이터 분포 자체가 다르기 때문** — Phase 2에서 이미 확인한 전세비율 시간 추세와 일관된 결과. 절대값을 직접 비교하기보다 "폴드 간 분산이 작다"는 안정성 결론에 집중할 것.

## Phase 6으로 넘어가도 되는가?

**최종 확정: QRF (onehot), Pinball 408.5(val) / 5-Fold CV 456.5±8.8(train 내 안정성 확인 완료) / Coverage@90 ~0.92.** 문제 없음 — Phase 6(모델 해석)으로 진행 가능.
